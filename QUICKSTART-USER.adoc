== What is Groove Browser Harness?

Groove Browser Harness is a Firefox extension that discovers
localhost services implementing the Groove protocol. It probes
well-known ports for services like Burble (voice), VeriSimDB
(database), Hypatia (scanner), PanLL (panels), and others, then
exposes their capabilities to web pages via `window.groove`.

No data leaves your machine. The extension only communicates with
`localhost` / `127.0.0.1` on specific ports.

[cols="1,1,2"]
|===

| 8+

This is a pure JavaScript browser extension. No build step needed.

== Install (Development Mode)

=== Method 1: Temporary Install (quickest)

1. Open Firefox
2. Navigate to `about:debugging#/runtime/this-firefox`
3. Click *"Load Temporary Add-on..."*
4. Select the `manifest.json` file in this repo

The extension loads immediately. It disappears when Firefox restarts.

=== Method 2: web-ext (auto-reload on changes)

[source,bash]
----
cd ~/Documents/hyperpolymath-repos/groove-browser-harness
web-ext run
----

Opens a fresh Firefox profile with the extension loaded. Reloads
automatically when you edit files.

== First Run

1. After loading, click the Groove Harness icon in the toolbar
2. The popup shows all 10 known Groove targets with their status
3. Green dot = connected, grey dot = not found
4. Click *"Probe All"* to re-scan

For any service to show as connected, it must be running on its
assigned port and serving `GET /.well-known/groove` with a valid
JSON manifest.

== Using the Page API

When the extension is active, any web page can use `window.groove`:

[source,javascript]
----
// Wait for the API to be ready
window.addEventListener("groove:ready", async () => {
  // Discover all services
  const result = await window.groove.discover();
  console.log("Connected:", result.connected);

  // Get status of all services
  const status = await window.groove.status();

  // Find which service provides "voice"
  const voiceSvc = await window.groove.findCapability("voice");

  // Send a message to Burble
  const response = await window.groove.send("burble", {
    action: "status"
  });
});
----
