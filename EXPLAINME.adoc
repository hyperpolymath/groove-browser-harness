= groove-browser-harness -- EXPLAINME

== Type-safe localhost service discovery for Gossamer, Burble, Vext, and other groove-aware systems

[quote, manifest.json]
____
Type-safe localhost service discovery for Gossamer, Burble, Vext, and other
groove-aware systems. Discovers local services and bridges their capabilities
into the browser.
____

A Firefox extension (Manifest V2) that probes known localhost ports for
groove-aware services and exposes them to web pages via a content-script bridge.
See `0-AI-MANIFEST.a2ml` for the full port map and architecture overview.

**Caveat:** Localhost only -- no remote discovery. The extension probes a
fixed set of 10 known ports, not arbitrary port ranges.

[cols="1,2", options="header"]
|===
| Path | Proves

| `manifest.json`
| Firefox Manifest V2 config, declares all 10 known ports in `host_permissions`

| `background/groove-discovery.js`
| Background script that performs port probing / service discovery

| `content/groove-bridge.js`
| Content script that bridges discovered services into the page context

| `popup/popup.html` + `popup/popup.js`
| Browser-action popup showing live service status
|===
